防火防烟分区与分隔

2019-06-23

防火防烟分区与分隔

在建筑物内划分防火分区,可有效地控制火势的蔓延,有利于人员安全疏散和扑救火灾,从而达到减少火灾损失的目的

防火分区

防火分区是指采用具有较高耐火极限的墙和楼板等构件作为一个区域的边界构件划分出的.

厂房的防火分区

  • 甲类厂房除因生产工艺需要外,应尽量采用单层建筑

  • 厂房设置自动灭火系统时,每个防火分区的最大允许建筑面积可按规定增加1.0倍.

    仓库的防火分区

  • 库房防火分区之间的水平分隔必须采用防火墙分隔,不能采用其他分隔方式.

  • 甲乙类仓库类的防火分区之间应采用不开设门,窗,洞口的防火墙分隔,且甲类仓库应采用单层结构.

  • 甲乙类仓库不应附设在建筑物的地下室他半地下室内.

    民用建筑的防火分区

    总建筑面积大于20000㎡的地下或半地下商业营业厅,应采用无门,窗,洞口的防火墙,耐火极限不低于2.00好的楼板分隔多个建筑面积不大于20000㎡的区域.

    木结构建筑的防火分区

    建筑高度不大于18m的住宅建筑,建筑高度不大于24m的办公建筑或丁、戊类厂房(库房)的房间隔墙和非承重外墙可采用木骨架组合墙体.

    城市交通隧道的防火分区

    隧道内的变电站,管廊,专用疏散通道,通风机房及其他辅助用房等,应采用耐火极限不低于2.00h的防火墙隔墙和甲级防火门等分隔措施与车行隧道分隔.

    防火分隔

    原则

  • 同一建筑物内,不同的危险区域之间,不同用户之间,办公用房和生产车间之间,应进行防火分隔处理

  • 作为避难通道使用的楼梯间,前室和具有避难功能的走廊,必须受到完全保护,保证其不受火灾侵害并畅通无阻

防火分区分隔

目的: 采取防火措施控制火灾蔓延,减少人员伤亡和经济损失

  • 水平防火分区,采用一定耐火极限的墙,楼板,门窗等防火分隔物按防火分区的面积进行分隔的空间.
  • 垂直防火分区,用楼板做分隔构件

    功能区域分隔

  • 歌舞娱乐放映游艺场所即使设置自动喷水灭火系统,面积也不能增加.
  • 观众厅,会议厅等人员密集场所,应设置火灾自动报警系统和自动喷水灭火系统等自动灭火系统,幕布的燃烧性能不应低于B1级.
  • 医院,疗养院建筑相邻护理单元之间应采用耐火极限不低于2.00h的防火墙分隔,隔墙上的门应采用乙级防火门,设置在走道上的防火门应采用常开防火门.
  • 住宅部分和非住宅部分采用耐火极限不低于1.50h的不燃楼板和耐火极限不低于2.00h且无门,窗,洞口的防火墙完全分隔
  • 商业服务网点的住`宅建筑,居住部分与商业服务网点之间应采用耐火极限不低于1.50h的不燃性楼板和耐火极限不低于2.00h且无门,窗,洞口的防火隔墙完全分隔.

    设备用房分隔

    附设在建筑内的消防控制室,灭火设备室,消防水泵房和通风空气调节机房,变配电室等,应采用耐火极限不低于2.00好的防火隔墙和不低于1.50h的防火隔墙和不低于1.50h的楼板与其他部位分隔.

    中庭防火分隔

  • 中庭应与周围相连的空间进行防火分隔
  • 高层建筑内的中庭回廊应设置自动喷水灭火系统和火灾自动报警系统
  • 中庭应设置排烟设施
  • 中庭内不应布置可燃物

    玻璃幕墙防火分隔

    玻璃幕墙在火灾初期即会爆裂,导致火灾在建筑物内蔓延,垂直的玻璃幕墙和水平楼板,隔间的缝隙室火灾扩散的途径
    防火措施的几方面的要求

  1. 对于不设窗间墙的玻璃幕墙,应在楼板外沿,设置耐火极限不低于1.0h,高度不低于1.2m的不燃性实体墙或防火玻璃墙,当室外设置自动喷水灭火系统时,该部分墙体的高度不应小于0.8m

  2. 水平缝隙和垂直缝隙应该用防火封堵材料严密填实.

    管道井防火分隔

    井道防火分隔要求

  3. 独立设置

  4. 严禁敷设可燃气体和甲乙丙类液体管道.

  5. 孔洞用防火材料或不燃材料严密填实.

    变形缝防火分隔

    采用防火封堵材料封堵

    管道空隙防火封堵

    采用防火封堵材料封堵

    防火分隔设施与措施

    防火分区的划分时通过防火分隔构件实现的

    防火墙

    具有不少于3.00h耐火极限的不燃实体墙

六个要求

  1. 直接设置在基础上或钢筋混凝土框架上,截断可燃性或难燃性墙体的屋顶结构,高出不燃性墙体屋面不小于40cm,高出可燃性墙体不小于50cm
  2. 建筑物外墙为难燃性墙体时,防火墙应突出墙的外表面40cm,或宽度每侧不小于2m.
  3. 防火墙距天窗大于4m时,且天窗端面为可燃材料,应采取防止火势蔓延的措施
  4. 防火墙不设排气管和门窗洞
  5. 不设在转角处
  6. 考虑屋架,梁,楼板等受到火灾的影响而破坏时,不致使防火墙倒塌

    防火卷帘

    分类
  • 按叶板厚度分轻型,重型
  • 按卷帘动作分上卷和侧卷
  • 按材料分普通型钢质,复合型钢质,非金属的复合防火卷帘

设置要求

  1. 替代防火墙的防火卷帘应符合防火墙的耐火极限的判定条件
  2. 在疏散走道的,具有延时下降的功能,电动手动控制
  3. 需要自动降落的,应具有信号反馈功能
  4. 应有防火防烟措施
  5. 不宜采用侧式防火卷帘
  6. 耐火极限不能低

设置部位

  • 设置防火墙有困难的部位

防火门窗

防火门

按耐火极限分 甲乙丙三级的耐火极限不低于1.50h,1.00h,0.50h,对应防火墙,疏散楼梯门,竖井检查门.
按材料,木质,钢质,复合材料防火门
按门扇结构: 带亮子,不带亮子,单扇,多扇

要求

  1. 应向疏散方向开启,关闭后应能从任一侧手动开启
  2. 用于疏散走道,楼梯间和前室的防火门,应能自动关闭
  3. 除了允许设置常开防火门的位置外,其他位置的防火门均应采用常闭防火门.
  4. 设在变形缝的,门开启不应跨越变形缝,防止烟气通过变形缝蔓延.
  5. 平时关闭具有防烟功能

防火窗

  • 耐火极限和防火门相同
  • 固定窗扇不能开启,平时采光,遮风挡雨,发生火灾时阻止火势蔓延
  • 活动窗扇起火时自动关闭,开启时排除烟气,设手动和自动开关.

    防火分隔水幕

    起防火墙的作用,在防火墙安装不了的地方安装

    防火阀

    在管道内阻火的活动式封闭装置,平时开启,挡管道内烟气温度大于70℃,自动关闭

设置部位

  • 穿越防火分区处
  • 穿越通风,空气调节机房的房间隔墙和楼板处
  • 穿越重要或火灾危险性大的房间隔墙和楼板处
  • 穿越防火分隔处的变形缝两端
  • 竖向风管与每层水平风管交接处的水平管段上.
  • 公共建筑的浴室,卫生间和厨房的竖向排风管
  • 公共建筑内厨房的排油烟管道安装150℃的防火阀

设置要求

  • 靠近防火分区
  • 方便维护的检修口处
  • 两侧2.0m的风管及其绝热材料应采用不燃材料
  • 符合国家标准

    排烟防火阀

  • 安装在排烟系统管道上,起隔烟,阻火作用的阀门
  • 具有手动自动功能
  • 满足耐火稳定性和耐火完整性的要求
  • 当管道内的烟气达到280℃,排烟防火阀自动关闭
  • 安装在排烟管进入排风机房处,穿越防火分区的排烟管道上,排烟系统的支管上

    防烟分区

    采用挡烟设施分隔而成

目的

  • 将烟气控制在一定范围内

  • 提高排烟口的排烟效果

    防烟分区面积划分

    要求

  • 防烟分区应采用挡烟垂壁,隔墙,结构梁等划分

  • 防烟区不应跨过防火分区

  • 防烟分区的建筑面积不能超过规范要求

  • 采用隔墙等形成封闭的分隔空间时,该空间宜作为一个防烟分区

  • 储烟仓高度不应小于空间净高的10%,且不小于500mm

  • 有特殊用途的场所应单独划分防烟分区

防烟分区分隔措施

  • 挡烟垂壁,垂直安装在建筑顶棚,横梁或吊顶下.
  • 分为固定和活动式,活动式可从初始位置自动允许到挡烟位置
  • 建筑横梁的高度超过50cm,可做挡烟设施用.

总平面布局和平面布置

2019-06-23

总平面布局和平面布置

建筑消防安全布局

建筑的总平面布局应满足城市规划和消防安全的要求

建筑选址

  • 周围环境,要考虑周围环境的相互影响,既要考虑本单位的成本,又要考虑邻近的企业和居民的安全

  • 地势条件,甲乙丙类液体仓库,宜布置在地势较低的地方

  • 主导风向,散发可燃气体,可燃蒸汽和可燃粉尘的车间,装置等,宜布置在明火或散发火花地点的常年主导的下风或侧风向.

    建筑总平面布局

    • 合理布置建筑,建筑之间留有足够的防火间距,以清除或减少建筑物之间及周边环境的相互影响和火灾危险.
    • 合理划分功能区域,尽量将火灾危险性相同的或相近的建筑集中布置.

建筑防火间距

防火间距是一座建筑物着火后,火灾不会蔓延到相邻建筑物的空间间隔,他是针对相邻建筑间设置的.

防火间距的确定原则

火灾时建筑物可能产生的热辐射强度是确定防火间距应考虑的主要因素.

  • 防止火灾蔓延,根据建筑的实际情形,将一二级耐火等级多层建筑之间的防火间距定为6m.
  • 保障灭火救援场地需要,规定一二级耐火等级高层建筑之间的防火间距不应小于13m
  • 节约土地资源,如果设定的防火间距过大,会造成土地资源的浪费.
  • 防火间距的计算,应按相邻建筑物外墙的最近距离计算.

防火间距

  • 乙类厂房与重要公共建筑的防火间距不宜小于50m;与明火或散发火花的地点的防火间距不宜小于30m

防火间距不足时的消防技术措施

  1. 改变建筑的生产和使用性质,降低火灾危险性,改变房屋耐火性能,提高耐火等级
  2. 调整生产厂房的部分工艺流程,限制库房内储存物品的数量,提高部分构件的耐火极限和燃烧性能
  3. 将建筑物的普通外墙改造为防火墙或减少相邻建筑的开口面积
  4. 拆除部分耐火等级低,占地面积小使用价值低且与新建建筑物相邻的陈旧建筑
  5. 设置独立的室外防火墙

    建筑平面布置

布置原则

  1. 内部着火,限制火灾和烟气在建筑内部和外部蔓延,并为人员疏散,消防人员的救援和灭火提供保护
  2. 内部发生火灾时,减少对邻近分隔区域的热辐射
  3. 方便消防人员救援
  4. 防止对人员和贵重设备造成影响或危害

2019-06-22

安全疏散

安全疏散是确保火灾中人员的生命安全具有重要作用

安全疏散基本参数

人员密度计算

  • 办公建筑有会议桌的会议室不应小于1.80㎡,无会议桌的不应小于0.80㎡
  • 录像厅、放映厅的疏散人数应根据厅、室的建筑面积按1.0人/㎡计算，其他歌舞娱乐放映游艺场所按0.5人/㎡
  • 餐饮场所,有固定座位的场所,疏散人数按实际座位数的1.1倍计算,展览厅的疏散人数根据展览厅建筑面积按0.75人/㎡.

    疏散宽度指标

    百人宽度指标
$$百人宽度指标={{N}\over{At}}b$$

式中的

  • N-疏散人数(即100人)
  • t-允许疏散时间(min)
  • A-单股人流通行能力(平坡哦43人/min,阶梯地面为37人/min)
  • b-单股人流宽度,0.55~0.60m
    疏散宽度
  • 厂房疏散出口的最小净宽度不宜小于0.9m,疏散走道不宜小于1.4m,疏散楼梯的最小净宽度不宜小于1.1m.
  • 高层民用建筑的疏散宽度

人贵有自知之明

   老子三十三章,知人者智也,自知者明也.大多数人理解的是人应该知道自己有几斤几两,不要自满,要谦虚,要认识到自己的不足.其实这种解释极其狭隘,因为人的一生所追求的修养的最高境界,就是认识自己.

   人作为高级的智慧生物,肯定是不能满足于基本的生理需要的,必定要向各个方向去探索,不断的超越极限.人类文明的成果是向所有的方向进行探索的成果,可是有些成果很难被人所继承下来,特别是关于个人修养方面的.中国拥有深厚的文化底蕴,对于个人修养也从很多的角度做了阐述.我个人非常喜欢,”人贵有自知之明”这句话,我认为这是个人修养的终极目标.

   人的一生大多数的时间都是在和别人打交道,哪怕再孤僻的人也是如此,所以一个人对他人的认识的理解很实际的体现在他在社会中的适应程度.那些对此有天赋的人,或者造诣很深的人,注定会成为人上人,也就是现在流行所说的高情商的人.那些人一个个的都是人精,因为他们对人性的理解比一般人要深刻很多,而且对人的情绪变化极其敏感.所以他们总是能洞悉你内心的想法,可以让你觉得和他聊天如沐春风,也可以让你觉得无地自容,情绪崩溃.简单的讲,这也是人的一种高级思维活动,所以是由天赋和后天练习的结果.特别是在官场上混的人,一般都比普通人强点,因为一个官员的升或贬很大程度都是由他的直接领导决定的.所以他们无时无刻都在揣摩上意,自然比一般人会强点.每个人都是很难理解别人的,因为我们确实不懂,而且也知道不懂,一些人会花时间和精力去尝试理解别人,从而得到更多的所获.

   一千人心中有一千个哈姆雷特,这也是说明这部戏剧写的好,因为更接近真实.每个人在别人心中的形象也是完全不同的.每个人都认为自己是最懂自己的,这没有错,毕竟和自己相处的时间是最长.由于觉得自己很懂自己,反而从来都不想去了解自己,所以有自知之明才如此难得可贵.认识自己不是偷偷放了个屁,别人都不知道就自己知道,是你知道别人都不知道你放了个屁.如果别人都知道是你放的,唯独你自己以为别人都不知道,这就是严重的无自知之明了.

   可以用一个比喻解释什么叫自知之明,我们都是棋盘上的棋子,作为二维动物,我并不知道,我是在棋盘上的哪个坐标的.其他人也不知道,可是我知道我和他们的相对坐标.他们也有一个坐标系用来标记其他人,我们要找的就是我们在别人坐标系中我的坐标.这个坐标就是我们要找的自知之明.

   认识自己,不可能凭空认识,我们只能通过别人的反馈映射自己的内心来认识别人从而认识自己.首先感谢那些给我带来痛苦的人,我要将这些痛苦的每个细节都仔细的品味,然后用牙齿一点点的研磨吞下,警醒自己不要堕落到黑暗中.其次感谢那些阳光的人,你们擅长制造欢乐,我要学习你们的处世态度,让外界的伤害更难伤害我.

建筑材料燃烧性能等级判据的主要参数及概念

  • 材料.指单一物质均匀分布的混合物
  • 燃烧滴落物/微粒.在燃烧试验过程中,从试样上分离的物质或微粒
  • 临界热辐射通量.火焰熄灭处的热辐射通量或试验30min时火焰传播到的最远处的热辐射通量
  • 燃烧增长速率指数——FIGRA.试样燃烧的热释放速率与其对应时间比值的最大值,用于燃烧性能分级.
  • THR600s.试验开始后600s内试样的热释放总量.

建筑构件的耐火极限

建筑耐火等级要求

建筑构件的耐火极限是衡量建筑物的耐火等级的主要指标

  • 厂房,仓库的耐火等级,建筑面积,层数等与其生产或储存阿类型有着密不可分的关系,对于甲乙类生产或储存的厂房和仓库,由于其生产或储存的物品危险性大,因此这类生产场所或仓库不应设置在地下或半地下,而且这类场所的防火安全性能的要求也较之其他类型的生产和仓储要高,在设计,使用时都应特别加以注意.

  • 民用建筑的耐火等级应根据其建筑高度,使用功能,重要性和火灾扑救难度的确定

    1. 地下或半地下建筑和一类高层建筑的耐火等级不应低于一级
    2. 单,多层重要公共建筑和二类高层建筑的耐火等级不应低于二级

  • 建筑高度大于100m的民用建筑,其楼板的耐火极限不应低于2.00h,一二级耐火等级建筑的上人平屋顶,其屋面板的耐火等级分别不应低于1.50h和1.00h.

  • 一二级耐火建筑的屋面板应采用不燃材料,单屋面防水层可采用可燃材料

  • 二级耐火等级的建筑采用难燃性房间隔墙,耐火极限不应低于0.75h

  • 二级耐火等级的建筑内采用不燃材料的吊顶,其耐火极限不限

  • 建筑内预制钢筋混凝土构件的节点外露部位,应采取防火保护措施,且节点的耐火极限不应低与相应构件的耐火极限.

生产和储存物品的火灾危险性分类

2019-06-17

生产和储存物品的火灾危险性分类

生产的火灾危险性分类

生产火灾危险性是指生产过程中发生火灾、爆炸事故的原因、因素和条件以及火灾扩大蔓延条件的总和.

厂房的火灾危险性类别是以生产过程中使用和产出物质的火灾危险性类别确定的,物质火灾危险性是确定生产的火灾危险性类别的基础.

评定物质火灾危险性的主要指标

主要依据:理化性质

评定气体火灾危险性的主要指标

  1. 爆炸浓度极限范围,爆炸下限,容易与空气或其他助燃气体形成爆炸性气体混合物
  2. 自然点越低,火灾爆炸的危险性越大
  • 气体的比重和扩散性
  • 化学性质活泼
  • 带电性以及热膨胀性

  1. 气体化学活泼性越强,发生火灾爆炸的危险性越大

  2. 气体在空气中的扩散速度越快,火灾蔓延扩展的危险性越大

  3. 相对密度大的气体易聚集不散,遇明火容易造成火灾爆炸事故

  4. 易压缩液化的气体遇热后体积膨胀,容易发生火灾爆炸事故

    评定液体火灾危险性的主要指标

    闪点是评定液体火灾危险性的主要指标
    对于可燃液体,通常还用自燃点作为评定火灾危险性的标志

    液体火灾危险性的标志

  • 爆炸温度极限
  • 受热蒸发性
  • 流动扩散性
  • 带电性

评定固体火灾危险性的主要指标

熔点和燃点是评定其火灾危险性的主要标志参数

许多低熔点的易燃固体还有闪燃现象

  • 粉状可燃固体是以爆炸浓度下限作为标志
  • 遇水燃烧固体是以与水反应速度快慢和放热量的大小为标志
  • 自燃性固体物料师以其自燃点为标志
  • 受热分解可燃固体师以其分解温度作为评定标志
